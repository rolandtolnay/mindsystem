## Iteration tips

- Generate 3-4 variations and pick the best composition
- If it still looks too busy: "Reduce to 6 tiles. More white space between cards."
- If icons look too detailed: "Simpler icons. Single-color line art only."
- If fonts look wrong: "Use thinner, lighter font weight. Match Apple's typography exactly."
- To refine a good result: "Same layout, but increase the gap between tiles slightly and make the font weight thinner"
